# StratisIO - DevOps Orchestration Platform

A powerful platform for orchestrating DevOps workflows.

## Getting Started

To start the platform, use the one-click launcher scripts:

- **Windows (CMD):** Run `run.bat`
- **Windows (PowerShell):** Run `run.ps1`

These scripts will automatically:
1. Seed the database with test data.
2. Start the FastAPI backend (Port 8000).
3. Start the Next.js frontend (Port 3000).
4. Open the dashboard in your default browser.
